INPUT LAYER:
  User provides one of:
    a) Single SMILES string (text box)
    b) Drawn molecule (JSME editor)
    c) CSV file with SMILES column (batch mode)
    d) SDF/MOL2 file (structure file)

PREPROCESSING LAYER (RDKit):
  1. SMILES standardization (neutralization, tautomer normalization)
  2. Molecule validation (reject invalid SMILES gracefully)
  3. Compute 200+ RDKit molecular descriptors
  4. Compute Morgan fingerprint (ECFP4, r=2, 2048-bit)
  5. Compute MACCS keys (167-bit)
  6. Build molecular graph (atoms=nodes, bonds=edges, atom/bond features)
  7. Generate 2D coordinate PNG image (for CNN / Grad-CAM / heatmap overlay)
  8. Compute QED, logP, TPSA, HBD/HBA, MW, rotatable bonds, aromatic rings

MODEL LAYER (PARALLEL):
  Stream A — SMILES Transformer:
    - ChemBERTa-2 (seyonec/ChemBERTa-zinc-base-v1 fine-tuned on Tox21)
    - Output: 768-dim CLS embedding
    - 12-head classification layer (one per Tox21 assay)
    - Loss: BCE with class weights for imbalance

  Stream B — Multi-task GNN:
    - PyTorch Geometric: 4-layer AttentiveFP or GIN
    - Node features: [atomic_num, degree, hybridization, aromaticity, ring, formal_charge]
    - Edge features: [bond_type, conjugated, in_ring, stereo]
    - Readout: global mean+max pooling concatenated
    - Joint toxicity correlation loss (JLGCN-MTT style)
    - Output: 256-dim graph embedding → 12-head sigmoid
    
  Stream C — Descriptor Baseline:
    - LightGBM GPU (multi-label, 12 binary heads)
    - Input: 200 RDKit descriptors + 2048 Morgan bits + 167 MACCS bits
    - Provides free SHAP values via TreeExplainer

ENSEMBLE LAYER:
  - Weighted average of logits from all 3 streams
  - Weights tuned on held-out validation set (scaffold split)
  - Final output: 12 × probability + composite risk score
  - Conformal prediction wrapper (MAPIE): adds calibrated intervals

XAI LAYER:
  - SHAP TreeExplainer → descriptor importance rankings
  - Captum IntegratedGradients on GNN → atom-level attributions
  - Grad-CAM on ChemBERTa attention → token-level saliency
  - SMARTS alert scanner → 150+ structural toxicophores flagged
  - RDKit Draw.MolToImage + custom color dict → heatmap PNG

FEATURE LAYER:
  - UMAP embedding (precomputed) → query molecule nearest neighbor
  - ADMET-AI predictions (10+ ADMET properties)
  - Bioisostere substitution engine → de-risked SMILES variants
  - Tanimoto similarity search → similar toxic/safe compounds
  - LLM report generator (Claude API) → structured PDF output

MASKED LOSS (HANDLE MISSING LABELS — CRITICAL):
  def masked_bce_loss(logits, labels, mask):
      """
      logits: [batch, 12]
      labels: [batch, 12] — NaN where label is missing
      mask:   [batch, 12] — True where label exists
      """
      loss = F.binary_cross_entropy_with_logits(
          logits, labels, reduction='none'
      )
      masked_loss = loss * mask.float()
      return masked_loss.sum() / mask.float().sum()
